快慢指针

快慢指针是用来判定链表成环的常用操作。

其具体的流程是设置一个快指针fast每次前进两格，一个慢指针slow每次前进一格

当它们可以相遇时便说明链表成环，当出现NULL时说明不成环

快慢指针的一些细节

快慢指针的相遇

假设链表环的入口entry 距离 head有L1个节点

当快慢指针相遇时，慢指针在环中前进了L2个节点

环的长度是C

那么我们可以得到

  • 慢指针走过的节点数：$L1+L2$
  • 快指针走过的节点：$2*(L1+L2)$
  • 由于两个指针相遇，所以：$2(L1+L2) = L1 + L2 + nC$

由上述假设，我们可以得到如下：
$$
L1+L2 = nC
$$
由于$L1 = kc+b; \space b >= 0$，所以$L2 <= C$，即代表着还没等慢指针跑完一圈，就和快指针相遇了

循环的入口

由上述公式我们还可以得到：
$$
L1 = nC-L2 = (n-1)C + (C-L2)
$$
这说明，加入有一个指针从头节点除法，慢指针从相遇点出发，那么它们的最终相遇的位置在entry处。
